Secure • Writes to your Google Sheet via Apps Script

Google Sheet Setup — 1 min

  1. Create a new Google Sheet. Name it StudyMuseAI Feedback
  2. In Row 1, add these exact headers:
    Timestamp | Name | Class | Domain | Email | Feedback | Mood | UserID
  3. Extensions → Apps Script → paste the code from google-apps-script/Code.gs
  4. Deploy → New deployment → Type: Web app → Execute as: Me → Who has access: Anyone
  5. Copy the Web app URL → paste it in js/feedback.js at SCRIPT_URL
